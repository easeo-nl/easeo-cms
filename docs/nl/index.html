Een CMS dat doet wat het moet.
Meer niet.

Lichtgewicht PHP CMS voor MKB-websites. Geen database, geen frameworks, geen node_modules. Draait op elke shared hosting met PHP.

Bekijk op GitHub Download .zip
$ git clone https://github.com/easeo-nl/easeo-cms.git && php -S localhost:8000 router.php

Digitale Soevereiniteit

Jouw data, jouw server, jouw regels. Geen Amerikaanse cloud nodig.

🏠

Self-hosted

Draait op elke Europese hostingprovider. TransIP, Antagonist, Hostinger EU — jouw keuze.

🔒

Geen cloud dependencies

Geen AWS, geen Google Cloud, geen Cloudflare nodig. Alle data blijft op jouw server als JSON-bestanden.

🇪🇺

GDPR by design

Geen externe tracking tenzij je het aanzet. Geen third-party cookies. Cookie consent ingebouwd.

📦

Zero vendor lock-in

Geen Composer, geen npm, geen SaaS-abonnementen. Fork het, maak het van jezelf, draai het voor altijd.

Kenmerken

0

Zero dependencies

Geen Composer, geen npm, geen build tools. Upload de bestanden, klaar. Enige dependency is PHPMailer voor SMTP — zit erin.

{ }

JSON in plaats van MySQL

Alle data als JSON-bestanden. Geen database server nodig, geen migraties, geen ORM. Werkt op elke hosting met PHP.

⚡

Veilig by design

2FA, CSRF-tokens, bcrypt, rate limiting, account lockout, CSP headers, SVG sanitization. Security is geen plugin.

5m

Setup in 5 minuten

Upload, bezoek het domein, doorloop de wizard. Bedrijfsinfo, branding, content en admin account in vijf stappen.

SEO

SEO uit de doos

Auto-sitemap, JSON-LD structured data, meta tags, breadcrumbs, Open Graph. Geen plugin nodig.

i18n

Meertalig

Ingebouwd i18n-systeem met JSON-taalbestanden. Engels en Nederlands standaard. Voeg elke taal toe met één bestand.

WordPress vs. EASEO CMS

WordPress EASEO CMS
Database vereist MySQL / MariaDB Geen — JSON bestanden
Dependencies PHP + MySQL + plugins Alleen PHP
Installatie Database aanmaken, wp-config, wizard Upload + wizard
Updates Core + thema + 10-30 plugins git pull
Beveiliging Afhankelijk van plugins Ingebouwd: 2FA, CSRF, CSP
Laadtijd Database queries + plugin overhead Direct file reads
Data soevereiniteit Vaak cloud/SaaS plugins vereist 100% op jouw server
Ecosysteem 60.000+ plugins Handmatig uitbreiden
Complexe sites Webshops, membership, LMS MKB-websites, blogs

Architectuur

index.php              Homepage
blog.php / blog-post.php   Blog engine
contact.php            Contactpagina
pagina-router.php      Dynamische pagina's

includes/              Core PHP (content, auth, forms, media, mail, i18n)
beheer/                Admin panel (donker thema)
templates/             Herbruikbare frontend sections
lang/                  Taalbestanden (en.json, nl.json, ...)
data/                  JSON data bestanden (git-ignored)
